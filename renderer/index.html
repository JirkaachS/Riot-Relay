1 · Accounts

Every identity stays separate

Add login credentials once, then sync while that account is active. Riot ID and PUUID verification prevent ranks, inventories, portraits, and sessions from attaching to the wrong roster entry.

PUUID is always authoritative.
2 · Switching

Switch only—or launch a game

Switch session changes the Riot account without launching a game. The VALORANT, League, and TFT actions switch and launch explicitly. If Riot asks for 2FA, enter it in Riot Client; the manager waits without restarting it.

3 · Stats

One unified account view

VALORANT, League, and TFT ranks appear together. League rank uses an exact-identity OP.GG lookup and verified League Client data as fallback. The Chat workspace shows friends and messages for only the currently authenticated Riot account.

4 · Inventory

Browse, value, and export

Load supported inventories, sort League skins by rarity, inspect chromas, emotes, icons, wards, TFT arenas and finishers, estimate VALORANT VP or League RP value, and export data or collection images.

5 · Deceive

Control how friends see you

Choose Offline, Mobile, Away, or Online; optionally hide game activity while preserving parties. The chat helper works in League. VALORANT validates real backend contacts, so use the in-app controls there.

Restart Riot after enabling Deceive for the first time.
